About Mob Deals

Mob Deals is a lightweight South Yorkshire SIM-only comparison project. It is designed to help users shortlist providers responsibly without unsupported guarantees or fake live-update claims.

Current status: this is a static GitHub Pages implementation with manually maintained comparison data. A verified automation pipeline should be added only when source URLs, timestamps, validation and review logs are available.

Principles

  • Be honest about data freshness.
  • Link users to provider pages for final price confirmation.
  • Keep the site fast, accessible and understandable.
  • Publish methodology, privacy and terms information clearly.
