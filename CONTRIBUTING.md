# Contributing to Mob Deals

Thanks for helping improve Mob Deals.

## Reporting a deal correction

Please include:

1. Provider name
2. Source URL
3. Visible monthly price
4. Data allowance
5. Contract length
6. Roaming/fair-use notes, if visible
7. Date/time checked

Do not submit private retention offers, account-only offers, or unverifiable voucher codes.

## Content standards

- Do not claim a price is guaranteed unless the guarantee process is documented and active.
- Always include provider/source links for deal data.
- Use plain English and avoid misleading urgency.
- Keep accessibility and keyboard navigation intact.

## Technical standards

- Keep the site static and fast.
- Avoid third-party runtime dependencies unless there is a clear need.
- Preserve semantic HTML, labelled inputs, visible focus states, and no-JavaScript readability.
- Run the quality checks in `.github/workflows/quality.yml` before merging.
