Accessibility Statement

Mob Deals aims to meet WCAG 2.2 AA principles wherever practical for a static comparison site.

Current accessibility features

  • Semantic landmarks and headings.
  • Keyboard-visible focus states.
  • Skip link.
  • Labelled calculator inputs.
  • HTML table fallback for comparison content.
  • No dependency on JavaScript for reading core deal data.
  • Reduced-motion support.

Feedback

If you find an accessibility issue, please report the page, device/browser, assistive technology if relevant, and what went wrong.
